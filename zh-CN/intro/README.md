# ss-panel

ss-panel 是一个shadowsocks多用户的前端。

## 服务器要求

* PHP 5.6 or newer
* Web server with URL rewriting
* MySQL



## Supported Server

* [shadowsocks manyuser](https://github.com/mengskysama/shadowsocks/tree/manyuser)
* [shadowsocks-go mu](https://github.com/orvice/shadowsocks-go)
